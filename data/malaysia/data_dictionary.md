# Malaysia Housing Dataset - Column Dictionary

## Target Variable
- **Median_Price**: Median housing price in Malaysian Ringgit (RM)

## Features
- **Township**: Housing area/development name
- **State**: Malaysian state location
- **Median_PSF**: Median price per square foot (RM)
- **Type**: Property type (Terrace, Semi-D, Condo, etc.)
- **Tenure**: Freehold or Leasehold
- **Transactions**: Number of transactions recorded

**Data Period:** 2025 market data
**Total Observations:** 1,946 townships
**Geographic Coverage:** Malaysia-wide
